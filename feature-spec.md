# Спецификация фич: AI Logo Studio (рефакторинг)

## Пересмотр стратегии

Решено: не запускать отдельный Tracewell как основной продукт, а
**отрефакторить существующий AI Logo Studio** до состояния реального
платного продукта. Позиционирование — самый дешёвый генератор фирменного
стиля для малого бизнеса (не подписка, разовая низкая цена).

Vectorizer (бывший Tracewell/VTracer-движок) не выбрасывается — становится
фичей внутри продукта: бесплатный инструмент "загрузи набросок логотипа →
получи вектор" как лид-магнит в воронку к платному brand kit.

Бэкенд: **остаётся FastAPI/Python** — рефакторинг существующего кода
быстрее переписывания с нуля на TypeScript. Решение оптимизировано под
скорость выхода к реальной выручке, не под портфолио.

---

## Корневая проблема, которую чиним первой

Текущий генератор логотипов выдаёт "ерунду" — то есть сам алгоритм
генерации не даёт пригодного результата. Это нужно починить раньше любой
инфраструктурной работы, иначе упрощать инфраструктуру вокруг
неработающего ядра бессмысленно.

### Новый движок генерации (вместо текущего)
- Шаблонный подход: иконка + пара шрифтов + палитра, без дорогой
  AI-генерации картинок
- Библиотека иконок с лицензией, разрешающей коммерческое использование
  без обязательной атрибуции (Phosphor Icons, Tabler Icons — MIT)
- Curated пары шрифтов (Google Fonts) — не случайный подбор, а
  предустановленные удачные комбинации
- Палитры по правилам цветовой теории (комплементарные/аналоговые), не
  рандом
- Несколько вариантов на выбор, доработка (замена иконки/шрифта/палитры)

## Фаза 1 — Рефакторинг до рабочего MVP

### Что упрощаем в инфраструктуре
- Убираем Kafka + Zookeeper + Confluent — генерация шаблонным движком
  быстрая, синхронная обработка достаточна (или простая таблица `jobs` в
  Postgres со статусом, если нужен прогресс-бар на фронте)
- Убираем LocalStack — переходим на реальный Cloudflare R2 (S3-совместимый,
  дешевле AWS, без платы за egress)
- Сохраняем PostgreSQL + SQLAlchemy + Alembic — рабочая часть, не трогаем
- Next.js 15 фронт — оставляем, упрощаем флоу под новый продукт

### Vectorizer как встроенная фича
- Загрузка наброска/растрового лого → трассировка в браузере (VTracer/WASM
  в Web Worker, как и планировалось для Tracewell)
- Бесплатно, без аккаунта — используется как лид-магнит
- Результат можно сразу использовать как основу для brand kit генерации

### Брендкит (платная часть)
- Логотип в нескольких форматах (горизонтальный/вертикальный/только знак)
- Цветовая палитра (HEX/RGB/CMYK)
- Пара шрифтов (заголовок + текст)
- Аватарка для соцсетей, шаблон визитки
- Экспорт: SVG, PNG, PDF-брендбук с базовыми правилами использования

### Ценообразование
- Бесплатно: vectorizer-инструмент, превью brand kit с watermark
- Разовый платёж (низкая цена, например €9-15) за полный brand kit без
  watermark — без подписки, это и есть отличие от Looka/Tailor Brands
- Подписка — НЕ в v1, рассматривается позже для агентств/повторных
  пользователей

### Авторизация — минимальная для v1
- Гостевая покупка через Stripe Checkout, доставка результата по email
  / прямому скачиванию после оплаты
- Полноценный аккаунт — только в Фазе 2, когда есть что хранить
  (история генераций)

## Фаза 2 — Личный кабинет (после первых платежей)
- Полноценная регистрация (email/password или OAuth)
- История сгенерированных brand kit'ов, повторное скачивание
- Сохранённые пресеты (любимые палитры/шрифты)
- Командные/агентские тарифы (несколько брендов на аккаунт)

## Фаза 3 — Расширение
- API для агентств/SaaS-интеграций
- Больше форматов экспорта (AI, EPS)
- Платный vectorizer batch-режим (множественная загрузка)

## Сквозные вопросы

### Юридические
- Проверка лицензий всех иконок/шрифтов на коммерческое использование
- Предупреждение пользователю о необходимости проверки на trademark-
  конфликты для сгенерированных логотипов
- Немецкие требования к выставлению счетов (Kleinunternehmer/VAT) при
  росте оборота

### Технические заметки по рефакторингу
- Бэкенд: FastAPI/Python — рефакторинг, не переписывание
- Хостинг: можно начать без полного Docker Compose стека — упрощённый
  docker-compose с backend+frontend+postgres, без Kafka/Zookeeper/LocalStack
- Домен: tracewell.app зарезервирован под vectorizer-фичу/поддомен,
  основной продукт — под существующим доменом AI Logo Studio (если есть)
  или под новым именем — обсудить отдельно
